# 验收方式
## 1. clone 到本地
```
git clone git@github.com:Zack-Bee/modao.git
cd modao
npm start // 开发
npm run build // 构建代码
```
## 2. 在gitpage查看
[gitpage页面](https://zackbee.cn/modao/dist/index.html)
